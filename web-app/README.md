# NetGuardian Web App (SaaS MVP)
> 基于 Streamlit 的网络分析可视化平台

## 🚀 启动方式

1. **安装依赖**：
   ```bash
   pip install streamlit networkx scipy numpy matplotlib
   ```

2. **运行应用**：
   ```bash
   streamlit run app.py
   ```

## 📦 功能说明
- **本地模式**：所有计算在浏览器所在的本地机器完成，数据不上传云端。
- **可视化**：支持动态网络图、鲁棒性曲线绘制。
- **报告导出**：集成 CLI 的报告生成功能。
